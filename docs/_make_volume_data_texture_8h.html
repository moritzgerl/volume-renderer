Creates a 3D OpenGL texture from volume data.

Uploads the volume's voxel data to the GPU as a 3D texture for volume ray-casting. Determines the internal format (R8 or R16) based on the volume's bits per component.

The texture is configured with linear filtering and clamp-to-edge wrapping for proper volume rendering.

Parameters
textureIdThe identifier for this texture in Storage.
textureUnitThe OpenGL texture unit to bind to (e.g., GL_TEXTURE0).
volumeDataThe volume data containing voxels and metadata.
Returns
Texture object configured as a 3D texture with the volume data.
