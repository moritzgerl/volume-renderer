SsaoUpdater Factory::MakeSsaoUpdater ( Storage & storage)

Creates and initializes the SSAO updater for monitoring parameter changes.

Constructs an SsaoUpdater that monitors GuiUpdateFlags for changes to SSAO parameters and regenerates SSAO resources (kernel, noise texture) when needed. The updater automatically updates shader uniforms with new SSAO kernel samples and regenerates the noise texture when the kernel size changes.

Parameters
storageStorage containing GUI update flags, SSAO kernel, textures, and shaders.
Returns
Initialized SsaoUpdater object.
See also
SsaoUpdater for SSAO resource update implementation.
GuiUpdateFlags for parameter change tracking.
SsaoKernel for SSAO sample generation.
