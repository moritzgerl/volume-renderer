Detailed Description

Renderable unit cube for volume rendering ray entry/exit points.

Provides a cube primitive with vertices at (-0.5, -0.5, -0.5) to (0.5, 0.5, 0.5). Used in volume rendering to determine ray entry and exit points by rendering the cube's front and back faces.
