Detailed Description

Contains constant string literals for INI file section headers.

Provides compile-time constant string views for all section header names used in application state INI files. These constants are used for both parsing (comparing against input) and serialization (writing headers).

Regular sections use complete header strings (e.g., "[Camera]"), while indexed sections use prefixes (e.g., "[TransferFunctionPoint") that are concatenated with indices during serialization.

See also
ApplicationStateIniFileSection for section identifier enumeration.
GetApplicationStateIniFileSection for mapping strings to enum values.
SaveApplicationStateToIniFile for writing section headers.
