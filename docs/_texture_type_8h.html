enum class TextureType
strong

Identifies the dimensionality of OpenGL textures.

Specifies whether a texture is 1D, 2D, or 3D, which determines how it is created, sampled, and bound in OpenGL. Each type maps to a corresponding OpenGL texture target (GL_TEXTURE_1D, etc.).
