CameraParameters Factory::MakeDefaultCameraParameters ( )

Creates camera parameters with default view settings.

Constructs a CameraParameters object initialized with default values for camera position, target, up vector, field of view, and near/far clipping planes. The default configuration provides a standard view of the volume suitable for initial exploration.

Returns
CameraParameters object with default view settings.
