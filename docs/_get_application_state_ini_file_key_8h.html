Persistence::ApplicationStateIniFileKey Persistence::GetApplicationStateIniFileKey ( std::string_view key)

Converts a string key to an ApplicationStateIniFileKey enum value.

Maps INI file parameter key strings (e.g., "PositionX", "SsaoRadius") to their corresponding enum values for type-safe parameter identification. Returns ApplicationStateIniFileKey::Unknown if the key string is not recognized.
