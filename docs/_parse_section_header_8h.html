Persistence::ApplicationStateIniFileSection Persistence::ParseSectionHeader ( std::string_view line)

Parses a section header from an INI file line.

Extracts the section name from a line in the format "[SectionName]" or "[SectionName:Index]" (for indexed sections like point lights). Returns the corresponding ApplicationStateIniFileSection enum value, or ApplicationStateIniFileSection::None if the section is not recognized.

The parsed section identifier.
