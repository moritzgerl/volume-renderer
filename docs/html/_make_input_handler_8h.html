InputHandler Factory::MakeInputHandler ( Storage & storage)

Creates and initializes the input handler for keyboard and mouse input.

Constructs an InputHandler that processes user input from keyboard and mouse to control camera movement, GUI visibility, and debug display modes. The input handler registers GLFW callbacks for the window and updates camera and display properties in response to user interaction.

Parameters
storageStorage containing camera, display properties, and GLFW window.
Returns
Initialized InputHandler object.
