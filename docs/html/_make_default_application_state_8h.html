Persistence::ApplicationState Factory::MakeDefaultApplicationState ( )

Creates application state with default values for all components.

Constructs an ApplicationState object initialized with default values for camera parameters, GUI parameters, and transfer function control points. This provides a consistent starting state for the application when no persisted state file is available.

Returns
ApplicationState object with default values.
