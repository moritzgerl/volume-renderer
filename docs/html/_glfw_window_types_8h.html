using Context::WindowPtr = std::unique_ptr<GLFWwindow, GlfwWindowDeleter>

Smart pointer type for GLFW window management.

Type alias for std::unique_ptr with a custom deleter that properly destroys GLFW windows. This provides automatic resource management for GLFW windows following RAII principles.

The custom deleter ensures glfwDestroyWindow is called when the unique_ptr goes out of scope or is reset.

See also
GlfwWindowDeleter for the custom deleter implementation.
GlfwWindow for the window wrapper class using this type.
