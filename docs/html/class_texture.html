Detailed Description

Encapsulates OpenGL texture creation, binding, and management.

Supports creation of 1D, 2D, and 3D textures with various internal formats, filtering, and wrapping modes. Each texture is bound to a specific texture unit and identified by a TextureId for type-safe retrieval from Storage.

Textures are created via Factory::MakeTextures() which configures all textures needed for the rendering pipeline including volume data textures, transfer function textures, framebuffer attachments, and SSAO noise textures.

See also
TextureId for the enumeration of texture types.
TextureType for distinguishing 1D, 2D, and 3D textures.
Factory::MakeTextures for construction of all textures.
TextureUnitMapping for named texture unit assignments.
RenderPass for binding textures during rendering.
