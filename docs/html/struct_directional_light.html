Detailed Description

Parameters for a directional light source.

Represents a light source at infinite distance with uniform direction, such as sunlight. Contains ambient, diffuse, and specular color components along with intensity and direction.

Default values are provided via Factory::MakeDefaultDirectionalLight(). Parameters can be adjusted at runtime through the GUI.

See also
Factory::MakeDefaultDirectionalLight for default light configuration.
GuiParameters for storing directional light parameters.
UpdateLightingParametersInShader for passing light data to shaders.
