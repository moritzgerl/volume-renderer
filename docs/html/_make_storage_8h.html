Storage Factory::MakeStorage ( )

Creates and initializes the central storage system with all application resources.

Constructs the Storage object containing all application state and OpenGL resources including camera, display properties, GUI parameters, primitives (screen quad, unit cube), SSAO kernel, textures, shaders, framebuffers, volume data, transfer function, and GLFW window. All components are initialized in the correct dependency order. This is the primary entry point for application initialization.

Returns
Fully initialized Storage object containing all application resources.
