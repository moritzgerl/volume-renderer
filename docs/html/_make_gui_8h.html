Gui Factory::MakeGui ( Storage & storage)

Creates and initializes the ImGui-based graphical user interface.

Constructs a Gui object that provides runtime controls for adjusting rendering parameters including lighting (directional and point lights), SSAO settings (kernel size, radius, bias), and transfer function control points. The GUI modifies GuiParameters and sets GuiUpdateFlags to trigger resource regeneration when needed.

Parameters
storageStorage containing GUI parameters, update flags, transfer function, and window.
Returns
Initialized Gui object.
