Detailed Description

Parameters for a point light source.

Represents a light source at a specific position in 3D space that radiates in all directions. Contains ambient, diffuse, and specular color components along with intensity and position.

Default values are provided via Factory::MakeDefaultPointLights(). Parameters can be adjusted at runtime through the GUI.

See also
Factory::MakeDefaultPointLights for default light configuration.
GuiParameters for storing point light parameters.
UpdateLightingParametersInShader for passing light data to shaders.
