Detailed Description

Manages vertex coordinate data for a unit cube.

Provides vertex position data for a cube with vertices at (-0.5, -0.5, -0.5) to (0.5, 0.5, 0.5). Used in volume rendering to determine ray entry and exit points by rendering the cube's front and back faces.

The cube is defined as 36 vertices (6 faces × 2 triangles × 3 vertices), with each vertex containing 3D position coordinates.

See also
UnitCube for the renderable cube primitive using this data.
VertexBuffer for uploading vertex data to the GPU.
