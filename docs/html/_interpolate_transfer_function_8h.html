glm::vec4 InterpolateTransferFunction ( float normalizedValue,
std::span< const TransferFunctionControlPoint > activePoints )

Evaluates the transfer function at a given normalized value.

Interpolates between control points to generate smooth color and opacity transitions. Uses Catmull-Rom spline interpolation for opacity (smooth curves) and linear interpolation for color (to preserve color accuracy).

This function is called repeatedly to generate a 1D texture that maps scalar values to RGBA colors for volume rendering.

Parameters
normalizedValueScalar value in range [0, 1] to evaluate.
activePointsSpan of active control points, sorted by value.
Returns
Interpolated RGBA color as vec4(rgb, opacity).
