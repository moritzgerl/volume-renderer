◆ ShaderId

enum class ShaderId
strong

Identifies shader programs for different rendering stages.

Each shader ID corresponds to a compiled and linked shader program used by a specific render pass. Shaders are created via MakeShaders and stored in Storage for retrieval by render passes.

Shader for shader compilation and management.
MakeShaders for shader program creation.
