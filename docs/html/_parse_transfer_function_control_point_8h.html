Parses and applies a transfer function control point parameter.

Converts the string value to the appropriate type based on the parameter key and updates the corresponding field in the TransferFunctionControlPoint structure. Supported parameters include scalar value, color components (ColorR, ColorG, ColorB), and opacity. Returns an error if the value cannot be parsed or is invalid.

Parameters
keyThe parameter key identifying which control point field to update.
valueStringThe string value to parse.
pointThe transfer function control point to update.
Returns
Expected containing either void on success or an error code on failure.
