Detailed Description

Stores dimensional and format information for volume datasets.

Contains all metadata necessary to interpret raw volume data files, including dimensions (width, height, depth), data format (components, bits per component), and physical spacing (scale factors for anisotropic datasets).

Metadata is typically loaded from a .dat file accompanying the raw volume data. The metadata is used to create 3D textures and properly scale the volume during rendering.

See also
LoadVolumeRaw for loading metadata from .dat files.
VolumeData for the actual volume voxel data.
MakeVolumeDataTexture for creating 3D textures from volume data.
