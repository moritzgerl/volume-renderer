Parses and applies a GUI parameter from an INI file value.

Converts the string value to the appropriate type based on the section, parameter key, and element index, then updates the corresponding field in the GuiParameters structure. Handles various parameter types including trackball settings, SSAO configuration, directional light properties, point light properties (indexed), and general rendering settings. Returns an error if the value cannot be parsed or is invalid.

Parameters
sectionThe INI file section containing the parameter.
keyThe parameter key identifying which GUI field to update.
elementIndexThe index for array elements (e.g., point light number).
valueStringThe string value to parse.
guiParametersThe GUI parameters structure to update.
Returns
Expected containing either void on success or an error code on failure.
