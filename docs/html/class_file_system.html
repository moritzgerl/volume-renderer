Detailed Description

Utilities for resolving file paths relative to the project root.

Provides platform-independent path resolution for loading resources (shaders, volume data, etc.) at runtime. The class automatically determines the project root directory from CMake-generated configuration and resolves relative paths accordingly.

The root directory is determined by:

  1. The LOGL_ROOT_PATH environment variable (if set)
  2. The logl_root constant from root_directory.h (generated by CMake)
  3. A relative path from the binary location (fallback)

This allows the same code to work whether run from the build directory or after installation.

See also
root_directory.h for CMake-generated root path constant.
