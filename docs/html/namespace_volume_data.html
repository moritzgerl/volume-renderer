VolumeMetadataKey VolumeData::GetVolumeMetadataKey ( std::string_view key)

Converts a string key to a VolumeMetadataKey enum value.

Maps metadata key strings from .dat files (e.g., "Width", "Height", "BitsPerComponent") to their corresponding enum values for type-safe metadata field identification. Returns VolumeMetadataKey::Unknown if the key string is not recognized.

VolumeLoadingResult VolumeData::LoadVolumeRaw ( const std::filesystem::path & rawFilePath)

Loads a raw volume file using metadata from a companion .dat file.

The .raw format produced by pvm2raw contains no header - it's pure binary voxel data. Metadata (dimensions, bit depth, scaling) is read from a companion .dat file with the same base name as the .raw file.

For example, given "volume.raw", this function looks for "volume.dat" containing the metadata.

Parameters
rawFilePathPath to the .raw file.
Returns
VolumeLoadingResult containing VolumeData on success, or VolumeLoadingError on failure.

VolumeLoadingResult VolumeData::LoadVolumeRaw ( const std::filesystem::path & rawFilePath,
const VolumeMetadata & metadata )

Loads a raw volume file using provided metadata.

The .raw format produced by pvm2raw contains no header - it's pure binary voxel data. Metadata must be provided externally via the metadata parameter.

This overload is useful when metadata is already known or comes from a different source than a .dat file.
