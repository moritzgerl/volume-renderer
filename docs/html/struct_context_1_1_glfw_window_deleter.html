Detailed Description

Custom deleter for std::unique_ptr managing GLFW windows.

Provides proper cleanup for GLFW window resources when used with std::unique_ptr. Calls glfwDestroyWindow to ensure the window is properly destroyed when the unique_ptr goes out of scope.

This enables RAII-style management of GLFW windows, ensuring that resources are automatically cleaned up without manual calls to glfwDestroyWindow.

See also
GlfwWindowTypes for the WindowPtr type alias.
GlfwWindow for GLFW window wrapper using this deleter.
