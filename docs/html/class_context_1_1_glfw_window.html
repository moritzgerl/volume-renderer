Detailed Description

RAII wrapper for GLFW window creation and lifecycle management.

Encapsulates a GLFW window with automatic initialization and cleanup. Provides methods for checking window state, swapping buffers, and polling events. Uses a custom deleter to ensure proper GLFW resource cleanup.

The window is created in the constructor with the dimensions specified in Config::windowWidth and Config::windowHeight. The OpenGL context is made current automatically.

See also
GlfwWindowDeleter for custom resource cleanup.
InitGlfw for GLFW library initialization.
Config for window configuration constants.
