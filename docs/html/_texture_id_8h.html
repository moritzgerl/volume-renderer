enum class TextureId
strong

Identifies textures for volume data, transfer functions, and SSAO.

Each texture ID corresponds to a texture resource created via MakeTextures and stored in Storage. Textures include volume data, transfer functions, G-buffer attachments, SSAO outputs, and noise textures.

See also
Texture for texture creation and management.
MakeTextures for texture initialization.
