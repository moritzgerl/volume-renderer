CameraParameters & cameraParameters )

Parses and applies a camera parameter from an INI file value.

Converts the string value to the appropriate type based on the parameter key and updates the corresponding field in the CameraParameters structure. Supported parameters include camera position (PositionX, PositionY, PositionZ) and zoom level. Returns an error if the value cannot be parsed or is invalid.

Parameters
keyThe parameter key identifying which camera field to update.
valueStringThe string value to parse.
cameraParametersThe camera parameters structure to update.
Returns
Expected containing either void on success or an error code on failure.
