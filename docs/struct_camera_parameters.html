Detailed Description

Holds serializable camera state for persistence.

Contains the essential camera parameters that can be saved to and loaded from an INI file. Used for application state persistence across sessions.

See also
Camera for the camera implementation that uses these parameters.
LoadApplicationStateFromIniFile for loading camera parameters from disk.
SaveApplicationStateToIniFile for saving camera parameters to disk.
