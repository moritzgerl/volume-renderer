Detailed Description

Manages vertex coordinate data for a full-screen quad.

Provides vertex position and texture coordinate data for a screen-aligned quad covering the entire viewport. The quad is defined in normalized device coordinates (NDC) from (-1, -1) to (1, 1).

Each vertex contains position (x, y) and texture coordinates (u, v), arranged as two triangles forming a rectangle.

See also
ScreenQuad for the renderable quad primitive using this data.
VertexBuffer for uploading vertex data to the GPU.
