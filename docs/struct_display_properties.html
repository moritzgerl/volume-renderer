Detailed Description

Stores flags for controlling debug visualization modes.

Contains boolean flags that control which debug visualizations are displayed in the rendering output. Modified by keyboard input via InputHandler and used by render passes to determine which content to display.

Currently supports toggling the SSAO map visualization, which displays the raw ambient occlusion values instead of the final composited image.

See also
InputHandler for keyboard input handling that modifies these flags.
RenderPass for using display properties to control rendering output.
Storage for centralized display properties storage.
